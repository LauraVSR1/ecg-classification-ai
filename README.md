# Clasificación de ECG con Inteligencia Artificial

Este proyecto implementa un modelo de Deep Learning para clasificar electrocardiogramas en normales y anormales.

## Tecnologías usadas
- Python
- TensorFlow / Keras
- Streamlit

## Modelo
Se utilizó DenseNet121 con transferencia de aprendizaje.

## Dataset
El dataset fue obtenido de Kaggle.

## Cómo ejecutar
1. Instalar dependencias:
pip install tensorflow streamlit

2. Ejecutar la app:
streamlit run app.py
